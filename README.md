## Description

[Twitch](https://twitch.tv/) webhook template TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# debug mode
$ npm run dev:debug
```

## Lint

```bash
# unit tests
$ npm run lint
```

## Test

```bash
# unit tests
$ npm run test
```
